Creación de un carrito de compras mediante JSP y MySQL.

Los productos se encuentran almacenados en la base de datos y son recorridos por la aplicación JSP para volcarlos en la página.

El aplicativo maneja uso de sesiones con usuarios de tipo cliente y administrador.

Según el perfil del usuario, se habilitan los módulos de Detalle Venta y Agregar Producto si es administrador.

En caso de ser usuario de tipo Cliente, puede añadir cuantos productos desee a su carrito y realizar el pago de los mismos mediante un botón PayPal.

Al final fue introducida a la página de bienvenida un estilo CSS el cual no es enfasis de programación en el proyecto.

Se hace uso de servlets, clases y métodos java 8.0.
